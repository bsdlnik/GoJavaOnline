Домашнее задание к Модулю 6

В лекции №3 OOP in Java заданием было написать код классов к диаграммам UML.
В данном ДЗ нужно:
	1. Дополнить код уже имеющихся классов обработкой исключительной ситуации. При этом использовать УЖЕ СУЩЕСТВУЮЩИЙ класс (например, IllegalStateException)
	2. Придумать СВОЙ класс исключения, относящийся к домену (задаче из лекции №3) и написать код по его генерированию и обработке.

Создать 2-3 проверямых исключения и добавить их обработку в классы из ДЗ к лекции №6. 
Создать аналогичные им непроверяемые исключения. 
Реализовать “обёртку” проверяемых исключений: в коде обрабатывается проверяемое исключение (конструкция try-catch), в результате чего помимо сообщения об ошибке генерируется соответствующее непроверяемое исключение. Непроверяемое исключение “пробрасывается” дальше.
